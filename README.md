Simple E-Commerce Website

A frontend e-commerce website built with vanilla HTML, CSS, and JavaScript. This project demonstrates core e-commerce functionality with a clean, responsive design.

✨ Features

· Product Display - Browse products in a grid layout · Shopping Cart - Add and remove items from cart · Cart Management - View cart summary with total price · Checkout Page - Simple checkout form interface · Responsive Design - Works on desktop and mobile devices

🚀 Tech Stack

· HTML5 - Structure and content · CSS3 - Styling and responsive design · JavaScript - Interactive functionality and cart management · LocalStorage - Cart persistence across page reloads

📁 Project Structure

├── index.html          # Home page with product listing
├── cart.html           # Shopping cart page
├── checkout.html       # Checkout page
├── style.css           # All styling
├── script.js           # JavaScript functionality
└── README.md
🛠️ How to Use

Clone the repository
git clone https://github.com/daniel001-beep/Ecommerce-Website.git
cd Ecommerce-Website.
Open the website · Simply open index.html in your browser · No build steps or dependencies needed!
Navigate the site · Browse products on the home page · Click "Add to Cart" to add items · View your cart by clicking on the Cart link · Proceed to checkout when ready
📱 Pages

· Home (index.html) - Product listing with add-to-cart buttons · Cart (cart.html) - View selected items and total price · Checkout (checkout.html) - Customer information form

🎯 Core Functionality

· Add items to cart · Remove items from cart · Update quantities · Calculate total price · Cart data persists using localStorage · Responsive navigation between pages
